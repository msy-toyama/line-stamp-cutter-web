# LINE Stamp Cutter

LINEスタンプ制作を効率化するWebアプリケーションです。
1枚の画像から複数のスタンプを自動で切り出し、背景透過やサイズ調整を行えます。

## ✨ 機能

- **画像スライス** - 1枚の画像から複数のスタンプを自動切り出し
- **グリッド編集** - ドラッグ操作でグリッド線を自由に調整
- **背景透過** - 自動背景除去、クリック透過、消しゴムツール
- **レイアウトプリセット** - 8/16/24/32/40個のクイック選択
- **トリミング・間隔調整** - リアルタイムプレビュー
- **ZIPダウンロード** - LINEクリエイターズマーケット仕様で出力

## 📐 出力仕様

- **スタンプ画像**: 370 × 320 px (PNG)
- **メイン画像**: 240 × 240 px (PNG)

## 🚀 使い方

1. 画像をアップロード
2. グリッドレイアウトを選択・調整
3. 必要に応じて背景を透過
4. ZIPでダウンロード

## 💻 ローカル実行

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
